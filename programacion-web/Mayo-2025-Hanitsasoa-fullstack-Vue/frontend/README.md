# Desarrollo Fullstack Node.js y Vue.js

## Introducción

Esta prueba técnica está diseñada para evaluar tus habilidades como desarrolladora fullstack, con énfasis en la implementación de una arquitectura cliente-servidor utilizando tecnologías modernas de JavaScript. Deberás crear una aplicación web sencilla que conste de un backend en Node.js y un frontend en Vue.js, los cuales se comunicarán entre sí a través de una API REST.
